## Hardware Overview

### Main PCB (my life-au.03c)
- Big black blob → main MCU / firmware
- SPY0030A → co-processor / input / IR
- AF0724 24C02 → EEPROM user data
- IR LED → connected to MCU
- Slot Magic Card → position known

### Secondary Board (my life key-02b)
- Dedicated to buttons / input
- 9 contacts → mapped to buttons (photos taken)
- Connected to main PCB

### Power / Batteries
- 3x AAA 1.5V
- VCC / GND routed to main PCB and secondary board
