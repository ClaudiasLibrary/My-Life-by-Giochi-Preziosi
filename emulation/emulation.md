# Emulation

## Current State of Emulation
As of today, there is no publicly available working emulator. The console is considered a closed proprietary system, which means:
- Lack of ROM dumps: the internal software dumps (data extractions) and the Magic Card data have not yet been published online.
- Unknown architecture: the processor is not documented. Many similar toys from the same era used Sunplus chips (a modified 6502 architecture), which require specific reverse engineering.
